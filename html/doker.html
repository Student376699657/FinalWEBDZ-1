Doker - платформа для управления контейнерами

Команды по работе с Doker

  • doker pull – Получить образ приложения
  • doker images – Вывести список образов
  • doker run -dp – Запустить контейнер с образом
  • doker ps – Вывести запущенные контейнеры
  • doker stop id_контейнера – Остановить контейнер
  • doker restart id_контейнера - Перезапустить контейнер
  • doker image rm -f id_образа - Удалить образ
